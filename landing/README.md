# Landing Page

Live page:

```text
https://vyahhi.github.io/tim/
```

Source in this repo:

```text
landing/index.html
```

To update the deployed page after editing the source:

```bash
git clone https://github.com/vyahhi/vyahhi.github.io.git /tmp/vyahhi.github.io
cp landing/index.html /tmp/vyahhi.github.io/tim/index.html
cd /tmp/vyahhi.github.io
git add tim/index.html
git commit -m "Update Tim landing page"
git push
```
